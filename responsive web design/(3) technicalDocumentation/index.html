Javascript introduction

JavaScript is a programming language used to create interactive and responsive websites, and can also be used for server-side programming and desktop app development. It's a versatile, high-level language that is widely used on the web.

Your first variable

Variables are very important in the world of coding they let us store values and manipulate these values easily lets create a variable!

remove the console.log and replace it with this

let message = "Hello world";

Well done you just declared your first javascript variable

when you hit run you will notice nothing happens this is because we arent outputing anything to the console to fix this add this underneath your code.

console.log(message);

When you hit run now you should see your message in the console!

Data types

There are 7 data types

  • boolean const x = true;
  • null const x = null;
  • undefined const x = undefined;
  • number const x = 12
  • string const x = "john doe"
  • object const x = { name: "peter" }
Variable types

There asre multiple variable types in javascript that all have uniuqe uses

The const variable is used to declare a variable that cant be changed

const x = 5;

x = 8;

//This will result in a error

The let variable is used to declare variables that can be changed

const x = 5;

x = 8;

//This will return 8

There is a also var variables this was used before const and let were introduced but are only used today for scoping problems they function like let variables

var x = 5;

x = 8;

//This will return 8
Manipulating variables

Variables can be manipulated to get new values

For example variables can be added

let x = 6;
x = 7 + 8;
console.log(x)
//15

You can also subtract, multiply (*) and divide (/) there are alot more but these are the basic ones!
